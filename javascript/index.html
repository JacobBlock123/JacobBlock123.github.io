Data Type Information
Strings

Strings Represent text data, strings can contain any character including letters.

Number

Number is an integer or floating point number, it can contain any number value including decimals.

Boolean

Boolean is a logical enitity that has two possible values: true and false.

Null

Null is used for an absence of any value, something set to null is intentionally empty of any value.

Undefined

Undefined is another empty value, but it differs from null in that it has simply not been assigned a value.

Symbol

Symbol is a unique datat type. They can't be converted to a string or to a number.

BigInt

BigInt is a numeric data type that is used to represent integers with arbitrary precision. A numeric value using the BigInt data type can surpass the safe integer limit.

Object

Object is a complex data type thats stores collections of data. It contains two properties: the data property and the accessor property.
